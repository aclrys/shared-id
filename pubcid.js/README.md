# Google ESP Collector Script

## Installation

```bash
npm install && npm run test
```

## Usage
Collector script URL: https://id.sharedid.org/lib/pubcid.min.js
```
<script>
	googletag.encryptedSignalSource['pubcid.org']();
</script>
```
